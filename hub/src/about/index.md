# About

这个网站很早之前就想做了，不过其实很早之前已经做了，第一次数据火葬场了，第二次做了个简易版本的，但是也没怎么维护，这次看到了 [驿站](https://yizhan.wiki/lezi-wiki/) 的网站，给了一些灵感，于是开始第三次重构。

网站名 Rakugen 是一言给的灵感，意思大概是 “乐子站” （ GPT 帮我起的 ）。

网站的初衷是用于记录一些乐子事件或是梗，为大家提供欢乐~~或治疗低血压~~，这不是所谓的「挂人」站，他们之间没有任何一点关系。「井蛙不可以语海，夏虫不可以语冰」，请文明玩梗，笑梗不笑人。本站仅供娱乐，不存在任何对任何人的攻击或者恶意。如果有任何疑义或提问，可通过相关方式联系我们。
